# AUMS_EXTENSION-

# Problem Statement

* Students currently have to navigate through multiple course pages in the AUMS portal to check for assignments.
* There is no centralized view that displays assignments from all courses in one place.
* Manually checking each course is repetitive, time-consuming, and inefficient.
* Important assignment deadlines can be missed due to the scattered nature of the information.
* The current workflow becomes increasingly inconvenient as the number of enrolled courses grows.

# Solution

* Develop a secure browser extension that integrates with the AUMS portal.
* Collect assignment information from all enrolled courses.
* Display assignment titles and due dates in a single popup interface.
* Reduce the time required to check pending assignments.
* Improve accessibility and provide a more user-friendly experience.

# Prototype Goals

* Display all assignments from enrolled courses.
* Show assignment due dates.
* Present the information in a clean and organized popup.

# Planned Future Updates

* Assignment deadline notifications.
* Custom reminders.
* Decode the Subject Code and Display Names.
* Search and filter functionality.
* Priority highlighting for upcoming deadlines.
* Calendar integration.
* Additional productivity features based on user feedback.
